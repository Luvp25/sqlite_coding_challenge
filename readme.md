Project Description:
    Comprehensive SQL-based data analysis suite designed to extract business intelligence from a relational database. I developed scripts to analyze customer lifetime value, categorize revenue by product type, and audit internal HR salary data to identify pay equity gaps.

Technologies Used:
    Database: SQLite3

    Editor: VS Code

    Extensions: SQLTools & SQLite Driver

    SQL Concepts: Common Table Expressions (CTEs),        Multi-table Joins, Aggregate Functions, and Data Sorting.

How to Run the Project:

    Ensure you have the bais_sqlite_lab.db file in the root directory.

    Open the challenge.sql file in VS Code.

    Using the SQLTools extension, create a new connection to the SQLite database file.

    Run the queries to see the results in the SQLTools results window.

What I Learned:

    Through this project, I moved from basic querying to professional-grade data analysis. My key takeaways include:

    Logical Refactoring: Learned how to use CTEs (Common Table Expressions) to make complex join logic more readable and maintainable.

    Data Integrity: Practiced using ROUND() and specific aliases to ensure financial data is presented in a business-ready format.

    Business Intelligence Thinking: Focused on writing queries that solve specific business problems, such as identifying the most valuable customers rather than just listing raw data.
